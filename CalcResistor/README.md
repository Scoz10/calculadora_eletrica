Este é um script simples em JavaScript desenvolvido para calcular o valor de resistência (em Ohms) de resistores baseando-se no código de cores universal. O programa suporta resistores de 4 e 5 faixas, calculando o valor nominal e a margem de tolerância.Shutterstock🚀 

Como FuncionaO script solicita ao usuário:A quantidade de faixas do resistor (4 ou 5).A cor de cada faixa sequencialmente.Com base nos dados inseridos, ele utiliza tabelas internas para definir:Dígitos Significativos: As primeiras faixas que formam o número base.Multiplicador: A potência de 10 pela qual o número base é multiplicado.Tolerância: A variação percentual permitida para cima ou para baixo.🛠️ FuncionalidadesCálculo de 4 Faixas: Considera (Dígito 1, Dígito 2) × Multiplicador ± Tolerância.Cálculo de 5 Faixas: Considera (Dígito 1, Dígito 2, Dígito 3) × Multiplicador ± Tolerância.Saída Dupla: Exibe os resultados tanto via alert() no navegador quanto no console.log().Precisão: Exibe os valores com até duas casas decimais.

📋 Exemplo de UsoAo rodar o código, você verá instruções no console e prompts na tela:Entrada: 4 faixas.Cores: MARROM, PRETO, VERMELHO, DOURADO.Processamento: * Base: $10$Multiplicador: $100$Tolerância: $5\%$Resultado: Valor fixo de $1000$ Ohm, variando entre $950$ e $1050$ Ohm.

🏗️ Como ExecutarCopie o código do arquivo .js.Abra o Console do Desenvolvedor no seu navegador (F12 ou Ctrl+Shift+I).Cole o código e pressione Enter.Siga as instruções nas janelas de prompt.[!IMPORTANT]Certifique-se de digitar o nome das cores em MAIÚSCULAS, exatamente como listado no menu do console (ex: VERMELHO, PRATA).
