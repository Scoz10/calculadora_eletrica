⚡ Conversor de Unidades: Base para MiliEste é um script simples em JavaScript desenvolvido para facilitar a conversão de grandezas elétricas da sua unidade base (como Volt ou Ampère) para a sua forma em mili (milésimos da unidade).

🚀 Como Funciona : O script utiliza janelas de interação direta no navegador (prompt e alert) para realizar o seguinte fluxo:Escolha da Unidade: O usuário define qual grandeza está utilizando ($A, V, \Omega$ ou $W$).

Entrada de Dados: O usuário digita o valor numérico na unidade padrão.Cálculo: O sistema aplica o multiplicador de $10^{3}$ (vezes 1.000).

Saída: O resultado é exibido em um alerta na tela e registrado no console do desenvolvedor.

📋 Exemplo de ConversãoSe você deseja converter 2 Volts para milivolts (mV):Entrada: 2Processamento: $2 \times 1.000 = 2.000$Saída: 2 V representam : 2000 mV
